# Dados (general vs return)

## general_reports

Base de todos os serviços protocolados no SIGOS.

Uso:

- acompanhamento operacional
- produtividade e execução
- volume por regional/equipe

## return_reports

Base de serviços em **status retorno**.

Uso:

- qualidade do processo em campo
- retrabalho
- auditorias e correções

!!! tip
    Separar as bases ajuda a criar dashboards com foco (operação vs qualidade).
